# Data-Viz Streamlit Deploy Skill

Generate complete `app.py` from `analysis-plan.md` chart assignments, fill in
chart code using templates, and deploy to Streamlit Cloud.

## Inputs

- `analysis-plan.md` — question → chart type → columns mapping
- `intake-summary.md` — project name, industry, dataset source
- `data/processed/clean_dataset.csv` — cleaned dataset
- `modules/chart-factory.py` — chart functions (from 06-workflow-gen)

## Code Generation Workflow

### Step 1: Read Chart Summary from analysis-plan.md

For each row in the Chart Summary table:
```
| # | Chart Type | X-Axis | Y-Axis | Color/Filter |
```

Map chart type → template file:
| Chart Type | Template |
|-----------|----------|
| bar | `templates/charts/bar-chart.py` |
| line | `templates/charts/line-chart.py` |
| scatter | `templates/charts/scatter-chart.py` |
| heatmap | `templates/charts/heatmap.py` |
| histogram | Use `px.histogram(df, x="{x_col}", title="{title}")` inline |
| box | Use `px.box(df, x="{x_col}", y="{y_col}", title="{title}")` inline |

### Step 2: Replace Placeholders in Templates

For each chart template, replace:
- `{x_col}` → actual column name from analysis-plan
- `{y_col}` → actual column name
- `{color_col}` → color/filter column (or remove parameter if none)
- `{title}` → the business question text

### Step 3: Assemble app.py

Use `templates/app-base.py` as skeleton. Fill in:

1. **Sidebar filters** — for each high-cardinality categorical column (>5 unique, <50 unique):
```python
selected_{col} = st.sidebar.multiselect("{col_label}", df["{col}"].unique())
if selected_{col}:
    df = df[df["{col}"].isin(selected_{col})]
```

2. **KPI metrics** — from analysis-plan.md KPIs section:
```python
col1, col2, col3 = st.columns(3)
col1.metric("{kpi_name}", f"{df['{col}'].{agg}():,.0f}")
```

3. **Chart sections** — pair charts 2-per-row:
```python
st.subheader("Section: {topic}")
col_a, col_b = st.columns(2)
with col_a:
    fig = bar_chart(df, x="{x}", y="{y}", title="{q1}")
    st.plotly_chart(fig, use_container_width=True)
with col_b:
    fig = line_chart(df, x="{x}", y="{y}", title="{q2}")
    st.plotly_chart(fig, use_container_width=True)
```

4. **Footer**:
```python
st.markdown("---")
st.caption(f"Data source: {dataset_source} | Generated by ck:data-viz")
```

### Step 4: Validate No Unfilled Placeholders

After generation, scan `app.py` for any remaining `{placeholder}` strings.
If found: fill them with actual values or remove unused code.

```python
import re
with open("app.py") as f:
    content = f.read()
unfilled = re.findall(r"\{[a-z_]+\}", content)
if unfilled:
    print(f"WARNING: Unfilled placeholders: {unfilled}")
```

## Deploy Instructions

### Local Preview
```bash
streamlit run app.py
# Opens http://localhost:8501
```

### Streamlit Cloud Deploy
1. Push project directory to GitHub repo (root must have `app.py` + `requirements.txt`)
2. Go to https://share.streamlit.io → "New app"
3. Select: repo, branch=`main`, entry file=`app.py`
4. Add secrets at Settings → Secrets (if BigQuery/Kaggle):
   ```toml
   BIGQUERY_CREDENTIALS_JSON = "..."
   KAGGLE_API_TOKEN = "..."
   ```
5. Click Deploy — auto-redeploys on every push to main
